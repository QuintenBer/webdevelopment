Opdracht 4

Open twee verschillende designs, rechtsklik en kies bij elk voor 'View page source'. Zet de twee vensters met HTML code naast elkaar op je scherm en vergewis je ervan dat de inhoud bij beiden (bijna) exact dezelfde is. Welke verschillen zie je?

De HTML code ziet er ongeveer hetzelfde uit. Ze hebben allemaal de normale opbouw van een HTML document met ook verwijzingen naar stylesheets. De ene pagina heeft soms een stylesheet minde rof meer dan de ander. Ook is de feitelijke inhoud anders omdat niet elke website gaat dezelfde inhoud gaat hebben. Maar iedere pagina heeft wel bijvoorbeeld linken, div elementen, lijst elementen, ...
